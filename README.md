Dental Training Simulator

## 📌 Overview

The **AI-Powered Dental Training Simulator** is an interactive 3D application developed using Unity that helps dental students practice clinical procedures in a virtual environment.

The system provides a **complete training cycle**:

* Practice with real-time guidance
* Test under exam conditions
* Viva for theoretical evaluation

---

## 🎯 Features

* 🦷 Two dental procedures:

  * Root Canal Treatment
  * Wisdom Tooth Extraction

* 🧪 Three training modes:

  * **Practice Mode** – Guided learning with AI feedback
  * **Test Mode** – Timed evaluation without hints
  * **Viva Mode** – Theory-based AI questioning

* 🤖 AI-driven system:

  * Real-time feedback
  * Error detection
  * Performance evaluation

---

## 🦷 Procedures

### Root Canal Treatment

1. Examine infected tooth
2. Drill to access pulp
3. Remove infected pulp
4. Clean the canal
5. Fill and seal

### Wisdom Tooth Extraction

1. Incision of gum
2. Loosening of tooth
3. Extraction
4. Cleaning and stitching

---

## 🧪 Modes

### Practice Mode

* Step-by-step guidance
* Highlighted working areas
* Real-time warnings for mistakes

### Test Mode

* No hints
* Timer enabled
* Performance scoring based on:

  * Accuracy
  * Time
  * Mistakes

### Viva Mode

* AI-generated questions
* Keyword-based answer evaluation

---

## 🧠 AI Techniques Used

* **Finite State Machine (FSM)** – Controls procedure flow
* **Rule-Based AI** – Validates user actions
* **Collision Detection** – Ensures precise tool interaction
* **Predictive AI** – Provides early warnings
* **Performance Analysis** – Generates final evaluation
